Why Lists and What?

A list refers to any information displayed in a logical or linear form. It is a series of items written together in a meaningful group or sequence and marked by bullet points, numbers, etc.

Types of lists in HTML

    There are 3 types of lists in HTML:
  • Unordered list or Bulleted list - < ul >
  • Ordered list or Numbered list - < ol >
  • Description list or Definition list - < dl >

An example is a shopping list:

  • milk
  • coffee beans

An example is a 2-nd shopping list:

  • bread
  • water

A numbered list is referred to as an “ordered list”

Ordered (numbered) lists are used to display a list of items that should be in a specific order. An example would be cooking instructions:

  1. Gather ingredients
  2. Mix ingredients together
  3. Place ingredients in a baking dish
  4. Bake in oven for an hour
  5. Remove from oven
  6. Allow to stand for ten minutes
  7. Serve
  8. Ask Hanna to try!

Ordered lists can be displayed with several sequencing options.

    Letters:
  1. Lowercase ascii letters (a, b, c…)
  2. Uppercase ascii letters (A, B, C…).
    Numbers:
  1. Decimal numbers (1, 2, 3…) - by default
  2. Decimal numbers with leading zeros (01, 02, 03…)
  3. Lowercase Roman numerals (i, ii, iii…)
  4. Uppercase Roman numerals (i, ii, iii…)

A common requirement in ordered list usage is to get them to start with a number other than 1 (or i, or I, etc.). This is done using the start attribute. This is useful if you have a single list of items, but need to break up the list with a note or other related information.
